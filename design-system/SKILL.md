Key non-negotiables:
- Colors are exact hex values — never substitute. See `colors_and_type.css`.
- Type is Cormorant Garamond (display) + DM Sans (body) only.
- Voice is French, editorial, *vous*-form, sentence-case headlines, no emoji.
- Mobile-first; property tiles are 2-column compact on mobile, never stacked.
- Wood warmth is bronze `#8B6F4E` everywhere (level 1); photographic wood texture at 10% opacity only on L'Aubade elements (level 2).
- Icons: Lucide, 1.5px stroke, bronze color.
